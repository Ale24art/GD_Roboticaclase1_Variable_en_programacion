Variables en Programación

Micro:bit · MakeCode · Fundamentos
❓ ¿Qué es una variable?

En programación, una variable se trata de un espacio en la memoria de un dispositivo que se utiliza para almacenar información de un determinado tipo de dato. Estas variables se identifican con un nombre único y sus valores pueden cambiar.

En palabras sencillas, una variable es un contenedor donde guardamos un dato que puede variar (cambiar) durante la ejecución del programa.

Las variables sirven para guardar y recuperar datos, representar valores existentes y asignar unos nuevos. Las variables permiten que los programadores se refieran a un valor con un nombre representativo en vez de tener que recordar este valor, lo que facilita muchas tareas complejas.

🧩 Componentes de una variable

Una variable en programación se caracteriza por:

  • Nombre único que la identifica dentro del programa.
  • Valor guardado que puede cambiar durante la ejecución del programa.
  • Tipo de datos que se almacena, como números, textos, estados booleanos, etc.
  • Alcance, que puede ser global, si se puede acceder desde cualquier parte del programa, o local, si solo se puede acceder dentro de un bloque de código.
👀 Tip del Docente

"Insista en que no llamen a las variables x o a. Deben llamarse puntos, velocidad, tiempo. Esto es una buena práctica de programación (limpieza de código)."

💻 Tipos de variables según los datos que almacena

Las variables pueden almacenar diferentes tipos de datos. Sin embargo, los más comunes son los siguientes:

  • Variables numéricas: Almacenan números enteros.
  • Variable de punto flotante: Guardan números con decimales.
  • Variables de caracteres: Representan símbolos o caracteres.
  • Variables de cadena de caracteres: Contienen palabras y textos.
  • Variables lógicas o booleanas: Sirven para almacenar valores lógicos (verdadero o falso).
🗂️ Tipos de Datos en MakeCode

Aunque MakeCode es visual, cuando se utiliza programación con bloques, maneja tipos de datos que debes distinguir:

Tipo Qué representa Cuándo usarlo Ejemplo en la vida real
Número Valores cuantitativos. Puntos, temperatura, coordenadas, cronómetros. La edad de una persona o el precio de un café.
Texto (String) Caracteres alfanuméricos. Nombres, contraseñas, mensajes de alerta, etiquetas. El nombre de un usuario o una dirección de correo.
Booleano Estados binarios. Interruptores, estados (encendido/apagado), validaciones. ¿Está la puerta abierta? (Sí/No).
¿Cómo elegir el tipo correcto? Pídales a sus estudiantes que se hagan estas preguntas:
  1. ¿Voy a hacer cálculos matemáticos con esto? → usa Número.
  2. ¿Es solo información para leer o mostrar? → usa Texto.
  3. ¿Es una respuesta de Sí o No? → usa Booleano.
🧮 Manipulación de Variables y Operadores

Manipular una variable significa cambiar su valor actual utilizando lógica o matemáticas. En MakeCode, esto se encuentra principalmente en la categoría Matemáticas.

Operadores Básicos:

  • Suma ( + ): Para acumular o avanzar (ej. subir de nivel).
  • Resta ( - ): Para reducir recursos (ej. perder una vida).
  • Multiplicación ( x ): Para cálculos rápidos (ej. duplicar puntos por un "power-up").
  • División ( / ): Para repartir o promediar (ej. calcular la media de tres notas).
"Concatenación": Explica que el símbolo + con texto no suma, sino que "pega" las palabras (ej. "Super" + "Micro" = "SuperMicro").
🛠️ Operaciones Fundamentales

Para dar una clase exitosa, debe dominar estos tres movimientos en la interfaz de MakeCode:

A
Declaración e Inicialización

Es crear la caja (variable) y darle un valor inicial. Generalmente se hace dentro del bloque "Al iniciar".

"Si no inicializa una variable, podría tener errores al intentar usarla."
B
Asignación (Fijar)

Es cuando reemplazas el contenido de la variable por uno nuevo.

Bloque: fijar [contador] a [0]

Lo que significa: "Borra lo que había antes y pone este nuevo dato, es decir establece ese dato como valor para la variable".

C
Actualización (Cambiar por)

Es cuando usas el valor actual para calcular el nuevo valor de la variable. Es vital para los contadores.

Bloque: cambiar [contador] por [1]

Lo que significa: Esto equivale a Contador = Contador + 1

⚠️ Error común: Los estudiantes suelen confundir el bloque Fijar (fijar un valor) con Cambiar (sumar/restar). Asegúrese de enfatizar esta diferencia.
